## When to use

Use this skill for daily note creation, review, journaling, and event-oriented workflows.

## Core patterns

- Use `daily today` or `daily show` to locate the relevant note first.
- Append structured entries rather than creating duplicate daily notes.
- Use `daily export-ics` when events need to leave the vault.

## Common mistakes

- Creating a second note for a day instead of appending to the indexed daily note.
