## When to use

Use this skill when external web material needs to be summarized or incorporated into notes.

## Core patterns

- Start with `web search`.
- Fetch only the pages that matter.
- Save the synthesis into a note rather than leaving the result ephemeral.

## Common mistakes

- Fetching many pages before narrowing the search.
